# combobox()

> 콤보박스 상태 관리 플러그인 — 열림/닫힘, 필터, 생성

## 커맨드

| 커맨드 | 설명 |
|--------|------|
| `open` | 콤보박스 열기 |
| `close` | 콤보박스 닫기 |
| `setFilter(text)` | 필터 텍스트 설정 |
| `create(label)` | 새 항목 생성 |

## 주요 export

| export | 설명 |
|--------|------|
| `comboboxCommands` | 콤보박스 커맨드 집합 |
| `COMBOBOX_ID = '__combobox__'` | 콤보박스 상태 키 |

## 의존

- core store utils

## 설계 원칙

- 모듈 레벨 상태 (isOpen, filterText)
