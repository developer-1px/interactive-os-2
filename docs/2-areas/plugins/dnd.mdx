# dnd()

> 키보드 드래그앤드롭 플러그인

## 커맨드

| 커맨드 | 설명 |
|--------|------|
| `moveUp(nodeId)` | 노드를 위로 이동 |
| `moveDown(nodeId)` | 노드를 아래로 이동 |
| `moveOut(nodeId)` | 부모 레벨로 이동 |
| `moveIn(nodeId)` | 이전 형제 안으로 이동 |
| `moveTo(nodeId, targetParentId, index?)` | 지정 위치로 이동 |

## 주요 export

| export | 설명 |
|--------|------|
| `dndCommands` | DnD 커맨드 집합 |
