# toolbar

> 수평 배치된 도구 버튼 모음

## 스펙

| 속성 | 값 |
|------|-----|
| role | toolbar |
| childRole | button |
| focusStrategy | roving tabindex |

## 축 조합

`composePattern(identity, ...axes)` 호출:
- activate(onClick)
- navigate(horizontal)

## 키맵 요약

| 키 | 동작 |
|---|------|
| ArrowRight | 다음 버튼으로 이동 |
| ArrowLeft | 이전 버튼으로 이동 |
| Home | 첫 번째 버튼으로 이동 |
| End | 마지막 버튼으로 이동 |
| Enter | 현재 버튼 활성화 |
| Space | 현재 버튼 활성화 |

## ARIA 속성

| 속성 | 값 |
|------|-----|
| aria-pressed | state.pressed |

## 관련

- UI: ⬜ (범용 컴포넌트 없음, ActivityBar에서 인라인 사용)
- 그룹: Navigation
