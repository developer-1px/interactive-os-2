# spatial

> DOM 좌표 기반 공간 탐색 (TV 리모컨 패러다임)

## 스펙

| 속성 | 값 |
|------|-----|
| role | group |
| childRole | group |
| focusStrategy | roving tabindex |
| selectionMode | single, multiple |

## 축 조합

`composePattern(identity, ...axes)` 호출:
- select()
- expand(enter-esc)
- activate(onClick)
- custom spatialNav (useSpatialNav)

> Note: 방향키가 DOM 좌표 기반으로 가장 가까운 요소를 탐색(useSpatialNav).

## 키맵 요약

| 키 | 동작 |
|---|------|
| ArrowDown | 아래쪽 가장 가까운 요소로 이동 |
| ArrowUp | 위쪽 가장 가까운 요소로 이동 |
| ArrowRight | 오른쪽 가장 가까운 요소로 이동 |
| ArrowLeft | 왼쪽 가장 가까운 요소로 이동 |
| Enter | 깊이 진입 (expand) |
| Escape | 상위 레벨로 복귀 |
| F2 | rename 모드 진입 |
| Space | 선택 토글 |

## ARIA 속성

| 속성 | 값 |
|------|-----|
| aria-level | depth level |
| aria-label | element label |
| aria-selected | state.selected |
