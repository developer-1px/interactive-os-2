# listbox

> 단일/다중 선택 가능한 수직 옵션 목록

## 스펙

| 속성 | 값 |
|------|-----|
| role | listbox |
| childRole | option |
| focusStrategy | roving tabindex |
| selectionMode | multiple, extended |

## 축 조합

`composePattern(identity, ...axes)` 호출:
- select(multiple, extended)
- activate(onClick)
- navigate(vertical)

## 키맵 요약

| 키 | 동작 |
|---|------|
| ArrowDown | 다음 옵션으로 이동 |
| ArrowUp | 이전 옵션으로 이동 |
| Home | 첫 번째 옵션으로 이동 |
| End | 마지막 옵션으로 이동 |
| Space | 현재 옵션 선택 토글 |
| Enter | 현재 옵션 활성화 |
| Shift+ArrowDown | 범위 선택 확장 (extended) |
| Shift+ArrowUp | 범위 선택 축소 (extended) |
| Ctrl+A | 전체 선택 (multiple) |

## ARIA 속성

| 속성 | 값 |
|------|-----|
| aria-selected | state.selected |
| aria-posinset | item index (1-based) |
| aria-setsize | total item count |

## 관련

- UI: ListBox
- 그룹: Navigation
