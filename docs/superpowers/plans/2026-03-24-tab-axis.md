# tab 축 신설 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 새로운 `tab()` 축을 만들어 4가지 Tab 전략(native, flow, loop, escape)을 제공하고, CMS 캔버스에 `tab('flow')`를 적용하여 Tab DFS 순회를 활성화한다.

**Architecture:** `tab()` 축은 `tabFocusStrategy` 전용 필드로 focusStrategy를 설정하고, `composePattern`이 `tabFocusStrategy ?? focusStrategy ?? DEFAULT` 순서로 최종 결정. navigate()의 focusStrategy와 순서 무관하게 tab()이 항상 우선한다.

**Tech Stack:** TypeScript, React, Vitest, @testing-library/react

- [ ] **Step 3: Implement tab() axis**

Create `src/interactive-os/axes/tab.ts`:

```ts
import type { BehaviorContext, FocusStrategy } from '../behaviors/types'
import type { AxisConfig, KeyMap } from './composePattern'

export type TabStrategy = 'native' | 'flow' | 'loop' | 'escape'

export interface TabOptions {
  orientation?: FocusStrategy['orientation']
}

/**
 * Tab navigation strategy axis.
 *
 * - `native`: engine does not intervene with Tab at all
 * - `flow`: all nodes tabIndex=0, browser Tab navigates in DOM order
 * - `loop`: like flow, but wraps last→first / first→last
 * - `escape`: roving-tabindex, Tab exits the zone
 *
 * **Warning:** Do not combine `tab('loop')` with `navigate({ grid: { tabCycle: true } })` —
 * both define Tab keyMap handlers and will conflict.
 */
export function tab(strategy: TabStrategy, options?: TabOptions): { keyMap: KeyMap; config: Partial<AxisConfig> } {
  if (strategy === 'native') {
    return { keyMap: {}, config: {} }
  }

  if (strategy === 'flow') {
    return {
      keyMap: {},
      config: {
        tabFocusStrategy: { type: 'natural-tab-order', orientation: 'both' },
      },
    }
  }

  if (strategy === 'loop') {
    const keyMap: KeyMap = {
      Tab: (ctx: BehaviorContext) => {
        const next = ctx.focusNext()
        // If focusNext returns same node, we're at the end — wrap to first
        if (next.payload?.nodeId === ctx.focused) return ctx.focusFirst()
        // Otherwise let browser handle normal Tab
        return undefined
      },
      'Shift+Tab': (ctx: BehaviorContext) => {
        const prev = ctx.focusPrev()
        // If focusPrev returns same node, we're at the start — wrap to last
        if (prev.payload?.nodeId === ctx.focused) return ctx.focusLast()
        // Otherwise let browser handle normal Shift+Tab
        return undefined
      },
    }
    return {
      keyMap,
      config: {
        tabFocusStrategy: { type: 'natural-tab-order', orientation: 'both' },
      },
    }
  }

  // escape — orientation configurable, defaults to 'both'
  const orientation = options?.orientation ?? 'both'
  return {
    keyMap: {},
    config: {
      tabFocusStrategy: { type: 'roving-tabindex', orientation },
    },
  }
}
```
